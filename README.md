## Project Description

Brief description of the system and the problem it solves.
T(n)-spector is an easy-to-use tool that automatically calculates code runtime, helping students overcome the struggle of measuring performance. It provides clear guidance on how to analyze execution speed and includes a shared database where users can save and access code snippets for future learning and comparison.

## Proposed Features
- Copy and paste a code snippet to calculate its runtime T(n) and/or O(n)
- Create code to check its current runtime
- Store the code and its runtime in a database for future purposes
